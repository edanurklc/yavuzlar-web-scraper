# EDDY - Web Scraper 

EDDY, Go dili kullanılarak geliştirilmiş, siber güvenlik odaklı platformlardan güncel haberleri ve duyuruları çeken web scraper aracıdır.

Seçilen web sitelerindeki başlık, tarih ve açıklama gibi verileri otomatik olarak ayrıştırır ve bunları düzenli `.txt` dosyaları halinde bilgisayarınıza kaydeder.



## 🛠️ Kurulum ve Gereksinimler

Projenin çalışabilmesi için bilgisayarınızda **Go (Golang)** yüklü olmalıdır.
